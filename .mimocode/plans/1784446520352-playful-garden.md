# Plan: Performance Mode - Arrow Keys Switch Scene

## Context
Performance mode dipakai untuk preview output di Android TV. Saat overlay menu hide, tombol arrow kiri/kanan harus bisa switch shader tab (scene).

## Change
Di `src/App.jsx` baris 149-156, tambah handling ArrowLeft/ArrowRight di block `performanceMode && !showOverlay`:

```js
if (performanceMode && !showOverlay) {
  if (e.key === 'Enter' || e.key === ' ') {
    e.preventDefault()
    setShowOverlay(true)
  } else if (e.key === 'Escape') {
    e.preventDefault()
    handlePerformanceToggle()
  } else if (e.key === 'ArrowLeft' || e.key === 'ArrowRight') {
    e.preventDefault()
    // Switch to prev/next tab
    const currentIdx = tabs.findIndex(t => t.id === activeTabId)
    const dir = e.key === 'ArrowRight' ? 1 : -1
    const nextIdx = (currentIdx + dir + tabs.length) % tabs.length
    setActiveTabId(tabs[nextIdx].id)
  }
}
```
