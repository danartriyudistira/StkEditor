# Syntetika Shader Editor

A web-based GLSL shader editor with live preview built with React + Vite.

## Features

- **Live Preview** - Real-time shader compilation and rendering
- **ISF Support** - Full ISF (Interactive Shader Format) compatibility
- **Monaco Editor** - Professional code editor with syntax highlighting
- **MIDI Control** - Map MIDI devices to shader uniforms
- **FX Chain** - Stack and chain multiple shader effects
- **WebSocket** - Real-time communication for external control
- **Transparency Control** - Adjust shader transparency levels

## Tech Stack

- React 19
- Vite
- Monaco Editor
- ISF (Interactive Shader Format)
- WebGL

## Getting Started

```bash
npm install
npm run dev
```

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+B` | Toggle editor |
| `Ctrl+.` | Toggle controls |
